## Tests
* You need Python and `requests` module to run tests. Examples (`/examples`) also used as test.
* Run all tests.
```
$ python test.py
```
* Run a single test. Test name is extensionless file name in `/examples` folder eg. hello, json.
```
$ python test.py <test_name>
```
